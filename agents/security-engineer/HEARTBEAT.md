# Heartbeat — Security Engineer

## Schedule
Spawned on demand by ATLAS. Periodic security posture checks when systems are active.

## First Action
Read the task context provided by ATLAS in the current session.

## Periodic Checks
- Vulnerability scan status: review latest scan results
- Compliance check: verify security controls in place
- Dependency audit: check for known vulnerabilities in dependencies
- Access review: verify permissions are least-privilege

## Status Reporting
Report security posture summary to ATLAS. Escalate Critical findings immediately.
