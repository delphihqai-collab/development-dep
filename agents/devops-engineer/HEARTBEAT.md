# Heartbeat — DevOps Engineer

## Schedule
Spawned on demand by ATLAS. Periodic health checks when infrastructure is active.

## First Action
Read the task context provided by ATLAS in the current session.

## Periodic Checks
- Infrastructure health: verify all deployed services responding
- Deployment status: check recent deployment outcomes
- Cost monitoring: review token/API usage against budgets
- Alert review: check monitoring dashboards for anomalies

## Status Reporting
Report infrastructure health summary to ATLAS. Flag any anomalies immediately.
