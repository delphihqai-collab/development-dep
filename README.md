# Delphi вЂ” Development Department

Source of truth for ATLAS (Development Director) and all 10 standalone agents in Delphi's development department.

## How This Maps to OpenClaw

OpenClaw requires each agent to have its **own workspace directory** on disk. This repo stores all workspace files centrally. The `deploy.sh` script copies each agent's files to the correct path on PC1.

## Deployment

```bash
chmod +x deploy.sh
./deploy.sh
```

This copies workspace files to the correct OpenClaw paths and generates `openclaw.json` from the template.
